Chrome 书签 ↔ 云书签 DB

                            
将当前页面服务器上的书签与 Chrome 书签进行双向同步。需通过 Chrome 插件触发，或使用下方按钮获取同步状态。

                            

                                
                                

                                    
                                

                            

                            

                            

                            

                                
💡 提示：点击 Chrome 插件弹窗中的「云书签双向同步」按钮，即可将 Chrome
                                    书签 ↔ Rainboard 云书签进行同步。

                                

                                    同步逻辑：
                                    
    
                                        
  • Chrome 有 → Rainboard DB 没有：将 Chrome 书签添加到 Rainboard
    • 
                                        
  • Rainboard DB 有 → Chrome 没有：将 Rainboard 书签添加到 Chrome
    • 
                                        
  • 两侧都有：跳过（幂等）
    •
