# Raindrop 前端对齐验收清单（Web）

## 1. 首页（工作台）

- 左侧账号卡片显示用户名/邮箱，点击可展开菜单（账号、设置、退出）。
- 快捷入口为图标化列表，计数正常显示，激活态明显。
- 已保存查询出现在侧栏，可点击应用，激活态与当前查询同步。
- 集合树支持展开/折叠、拖拽反馈、hover 操作按钮显示。
- 标签区支持排序切换、更多/收起、激活标签状态提示。
- 顶部工具栏在桌面端不换行错位，排序下拉宽度稳定。
- 高级筛选面板层级清晰（表单区/动作区/状态提示）。
- 视图切换支持行/卡片/标题/看板，布局偏好刷新后保留。

## 2. 列表/卡片/标题/看板视图

- 行列表默认展示；选中态、激活态、hover 态视觉一致。
- 列表列配置菜单可切换集合/类型/摘要/标签/时间显示。
- 卡片视图封面、摘要、操作按钮层级清晰。
- 标题视图 hover 后操作按钮出现，排版不抖动。
- 看板视图支持无封面条目触发首页预览抓取。
- 列表加载时出现骨架屏；空列表显示空态；接口失败显示错误态与重试按钮。

## 3. 右侧详情面板

- 默认查看态，点击编辑后进入编辑态；保存后退出编辑态。
- 顶部摘要卡片显示类型、标题、域名/时间、状态 chips。
- 抓取与阅读区显示摘要状态 chips（元数据/正文/预览状态）。
- 抓取详情与历史可折叠查看。
- 高亮列表为侧栏风格分组项，支持折叠查看注释。

## 4. 预览弹窗

- 头部信息区、工具条分组、状态 badge 层级清晰。
- 状态覆盖：未加载 / 加载中 / 已加载 / 降级预览 / 加载失败。
- 阅读模式按钮状态化（aria-pressed 与视觉态一致）。
- 预览状态会同步回详情摘要 chips。

## 5. 对话框与表单

- 新建书签/新建集合对话框具备头部说明、字段区、底部操作区。
- 表单校验使用内联红框与错误条提示。
- 取消/关闭按钮不触发必填校验。
- 移动端对话框可滚动，按钮布局不溢出。

## 6. 登录页 / 设置页 / 公开页

- 登录页为独立页面（未登录自动跳转），含品牌区与表单区。
- 设置页为左侧导航 + 右侧分组内容，支持锚点跳转与高亮。
- 公开页包含头部集合信息、计数、书签列表卡片与空/错态。

## 7. 可访问性与快捷键

- `Tab` 导航可达主要交互控件，focus ring 可见。
- 快捷键生效（`/`, `j`, `k`, `o`, `e`, `Delete`, `?`），输入框中不误触发。

## 8. 响应式

- 手机宽度下：顶部工具栏换行、对话框尺寸收缩、预览工具条分组可换行。
- 侧栏/主区/详情区在移动端顺序合理（侧栏 -> 主区 -> 详情）。
