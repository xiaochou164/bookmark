# Raindrop `/my/0` 交互回归清单（第二轮）

## 范围

- 工作台首页（`/` 对齐 `/my/0`）
- 侧栏、顶部搜索/工具栏、列表视图、详情抽屉、菜单/弹层、快捷键

## 1. 搜索与 suggestions

- 点击搜索框后打开 suggestions popover
- `↑/↓/Home/End` 能切换建议项焦点
- `Enter` 选择当前建议项
- `Esc` 关闭 suggestions（不影响页面状态）
- 输入普通关键词时采用防抖更新，不会每个字符都闪烁重载
- 输入后按 `Enter` 立即应用查询（不等待防抖）
- 失焦时若有 pending 搜索，会立即 flush

## 2. 顶栏与标题栏菜单

- `添加` split-button 主按钮可打开新建书签对话框
- `添加` 下拉菜单可打开（新建书签/创建集合/上传/导入导出）
- 标题栏 `排序` 菜单可打开并应用排序
- 标题栏 `视图` 菜单可切换列表/卡片/标题/看板
- 标题栏 `更多` 菜单可打开导入导出/同步插件入口
- 菜单在视口边缘不会溢出屏幕（锚点定位 + 边界吸附）

## 3. 侧栏集合树与菜单

- 点击集合切换过滤结果
- 集合展开/折叠正常
- 集合右键菜单和 `…` 菜单都可打开
- 集合树拖拽：
  - before（上插入线）
  - after（下插入线）
  - inside（行内高亮）
- 拖入折叠父集合中央区域时可悬停自动展开
- 拖拽结束后高亮态/插入线清理干净
- 不允许把集合拖到自身或自己的后代下（循环防护）

## 4. 系统集合 / 快速过滤 / 标签菜单

- 系统集合（全部/收件箱/收藏/归档/回收站）右键菜单可用
- 快速过滤行点击可应用过滤
- 快速过滤 `…`/右键菜单可打开（应用/复制/高级搜索/清除）
- 标签行点击可筛选标签
- 标签 `…`/右键菜单可打开（筛选/重命名/合并/管理/移除）

## 5. 列表视图行为

- 行 hover 显示完整动作按钮组（桌面端）
- 行点击激活详情抽屉
- 复选框 hover/选中显示逻辑正确
- `更多…` 增量加载可追加下一页（与分页不冲突）
- 大列表滚动时虚拟滚动正常（无明显白屏、错位、抖动）

## 6. 详情抽屉（Item Panel）

- 打开/关闭抽屉（关闭按钮 / 遮罩 / `Esc`）
- 顶部 `上一条/下一条` 切换条目
- `查看/编辑` 模式切换
- `网页预览/阅读模式` 按钮联动预览弹窗
- `更多` 菜单可打开（复制链接/预览/删除等）
- sections 折叠/展开状态刷新后保持
- 桌面端 `Tab/Shift+Tab` 在抽屉内循环（focus trap）

## 7. 快捷键

- `/`、`Cmd/Ctrl+K`：聚焦搜索框
- `j/k`：切换当前条目
- `o`：打开预览
- `p`：打开 Web 预览入口
- `e`：切换详情编辑态
- `f`：切换收藏
- `a`：新建书签
- `Shift+A`：新建集合
- `r`：刷新当前视图
- `Delete/Backspace`：删除当前条目（含确认流程）
- `?`：显示快捷键帮助提示

## 8. 层级与弹层（统一层）

- `search-suggest-popover`、`header-menu`、`context-menu` 层级不会互相压错
- dialog 打开时 backdrop 层级高于页面内容、低于 dialog
- 详情抽屉与 dialog 同时出现时，不会发生遮挡错乱

## 9. 回归记录格式（建议）

- 模块：`列表视图`
- 步骤：`滚动到底部 -> 点击更多… -> 再滚动`
- 结果：`出现重复条目`
- 严重级别：`P1/P2/P3`
- 关联待办：`RA-DOM-407/408` 或新建修复项
