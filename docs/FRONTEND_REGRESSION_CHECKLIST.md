# 前端回归清单（Web）

## 1. 认证与导航

- 未登录访问 `/` 自动跳转 `/login.html`
- 登录成功按 `next` 返回原页面
- 侧栏账号菜单可打开/关闭（点击外部、`Esc`）
- 设置页未登录跳回登录页

## 2. 首页核心流程

- 新建书签（成功 / 校验失败提示）
- 新建集合（成功 / 取消不触发校验）
- 切换快捷入口、集合、标签筛选
- 搜索与高级筛选（应用/重置/保存查询/删除保存查询）
- 布局切换（行/卡片/标题/看板）与刷新后保持

## 3. 列表与批量操作

- 勾选单项/多项，批量操作条显隐正确
- 批量收藏 / 归档 / 删除 / 移动
- 批量删除后撤销
- 行列表列配置切换后渲染正确
- 骨架屏、空态、失败重试可见且可用

## 4. 详情面板

- 查看态 -> 编辑态 -> 保存 -> 回到查看态
- 打开 / 预览 / 删除 / 恢复按钮
- 抓取状态摘要 chips 和折叠详情显示
- 高亮/注释 CRUD（含折叠项交互）

## 5. 预览弹窗

- 打开/关闭预览
- 阅读模式切换
- 提取正文
- 选区高亮（阅读模式）
- 加载失败与降级态提示

## 6. 导入导出与插件

- 导入导出对话框打开/关闭
- 任务列表刷新与结果展示
- 插件面板打开、预演、执行、任务状态轮询
- 调度器表单加载/保存/暂停/恢复

## 7. 设置页

- 左侧锚点导航点击滚动与高亮
- 账号资料保存
- Token 创建/吊销
- 会话刷新/吊销
- 共享邀请创建/删除/切换角色/接受
- 公开链接创建/启停/删除
- 套餐切换、配额刷新、备份创建/恢复、AI 占位接口按钮

## 8. 公开页与响应式

- `/public/c/:token` 正常加载公开集合
- 空集合/失败态显示正常
- 手机宽度下首页、预览、登录页、设置页无明显溢出

## 9. 快捷键与无障碍

- `/` 聚焦搜索框
- `j/k` 切换当前条目
- `o` 打开预览
- `e` 进入/退出详情编辑态
- `Delete` 删除当前条目（确认后执行）
- `?` 显示快捷键提示
