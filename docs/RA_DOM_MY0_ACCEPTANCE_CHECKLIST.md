# Raindrop `/my/0` 运行时 DOM 对齐验收清单（第二轮）

## 目标

- 以 `/my/0` 的运行时 DOM 与交互层级为参考，验收当前工作台在信息架构、组件层级、状态反馈上的对齐程度。
- 本清单关注“结构与交互”，不要求逐像素视觉复制。

## 1. SplitView 外壳（Sidebar / Main / Item Panel）

- 桌面端为左右分栏工作台结构（左侧栏 + 主内容区），详情为右侧覆盖式抽屉。
- 打开详情抽屉时主列表宽度不被重新压缩（覆盖式而非第三列挤压式）。
- 详情抽屉有遮罩层，点击遮罩可关闭。
- `Esc` 关闭顺序正确：先关 transient menu/popover，再关详情抽屉。
- 桌面端详情抽屉 `Tab/Shift+Tab` 焦点被圈定在抽屉内。

## 2. 侧栏（系统集合 / 集合树 / 快速过滤 / 标签）

- 左上账号入口显示用户名/邮箱，点击可展开菜单（账号 / 设置 / 退出）。
- 系统集合（全部书签、收件箱、收藏、归档、回收站）为 32px 密度行列表。
- 系统集合行含：图标、标题、计数、右侧 `…` 菜单；支持右键菜单。
- `My Collections` 分组头含 `…` 菜单，拖拽集合时分组头有拖拽高亮提示。
- 集合树支持：
  - 展开/折叠
  - 右键菜单与 `…` 菜单
  - 拖拽排序
  - 拖拽插入线（before/after）
  - 拖入子集合高亮（inside）
  - 悬停自动展开父集合
- 集合树数量较大时启用虚拟滚动，滚动过程无明显卡顿。
- 快速过滤为 section + 行列表，支持点击应用与右键/更多菜单。
- 标签区为行列表（非 pill），支持右键/更多菜单（筛选/重命名/合并/移除）。

## 3. 顶部搜索栏与主工具条

- 搜索框为主入口，左侧搜索图标，右侧内嵌高级搜索按钮。
- 搜索框 focus 时弹出 suggestions popover（建议项 / 最近搜索 / 帮助区）。
- suggestions 支持键盘导航：`↑/↓/Home/End/Enter/Esc`。
- 支持 token 建议（如 `tag:`、`type:`、`created:`、`note:true` 等）。
- 搜索输入有防抖，不会在每个按键都造成明显全量闪烁；`Enter`/失焦会立即应用。
- 顶栏 `询问`、`添加`（split-button）布局稳定，窄屏时文本可收缩但不溢出。
- `添加` 下拉菜单可打开，菜单项图标化且层级/hover 一致。

## 4. 集合标题栏（Collection Header）

- 标题栏显示当前视图图标、标题、打开源页面按钮（微按钮）。
- 标题栏右侧含排序 / 视图 / 更多（导出等）菜单入口。
- 标题栏与工具栏 sticky 层级正确，不互相遮挡。
- 打开多个菜单时层级和定位稳定（不会贴边溢出或被 header 遮挡）。

## 5. 列表视图（List）

- 默认视图为列表视图。
- 行结构接近 Raindrop：缩略图、标题、副信息、右侧动作区、选择框。
- 列表条目 hover/selected/active/focus 状态有明确层级。
- 右侧动作按钮集完整显示（打开、预览、Web、复制、询问、收藏、标签、编辑、删除/恢复）。
- 列表副信息密度合理（集合 / 域名 / 日期 / 类型）。
- 缩略图有加载占位、错误回退、无封面占位。
- 列表底部支持 `更多…` 增量加载，且与分页兼容。
- 大数据量时桌面列表启用虚拟滚动，滚动时不会频繁抖动重渲。

## 6. Item Panel（详情抽屉）

- 顶部工具条有上一条/下一条、查看/编辑模式切换、预览/阅读、更多、关闭。
- 查看态与编辑态切换明确，详情头部能反映当前模式。
- sections 支持折叠/展开，状态可持久化。
- 详情更多菜单图标化，菜单项 hover/danger 态统一。
- 键盘导航：
  - `ArrowUp/ArrowDown/ArrowLeft/ArrowRight` 切换条目
  - `Esc` 关闭菜单/抽屉

## 7. Popover / Menu / Dialog 统一层

- `header-menu`、`context-menu`、`search-suggest-popover` 使用统一视觉语言（背景/边框/阴影/圆角）。
- 菜单项高度、图标列、文本列、hover/focus/danger 态一致。
- 右键菜单/更多菜单采用锚点优先定位，边界吸附，避免超出视口。
- Dialog 与 Drawer 的 `z-index` 栈清晰，不会互相遮挡。

## 8. 快捷键与可访问性

- `/` 或 `Cmd/Ctrl+K` 聚焦搜索框。
- `j/k` 切换条目；`o/p` 打开预览；`e` 编辑；`f` 收藏；`a/Shift+A` 新建；`r` 刷新。
- `?` 显示快捷键提示。
- 菜单支持方向键导航；搜索 combobox 的 `aria-expanded/aria-activedescendant` 更新正确。
